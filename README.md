# Plan de formacion OpenAI Codex

Este repositorio contiene el plan de estudio completo para formar equipos de desarrollo en el uso de Codex como asistente/agente de programacion. El plan es modular, progresivo y pensado para aplicarse en entornos reales (CLI, IDE, app y cloud).

## Objetivos generales

- Entender el rol de Codex como agente que lee, edita y ejecuta tareas.
- Adoptar un flujo de trabajo seguro con revision de diffs y validacion.
- Estandarizar prompts, skills y configuraciones para trabajo en equipo.

## Modulos del plan

1. Fundamentos de Codex como agente de desarrollo.
2. Instalacion y uso eficiente de herramientas (CLI, IDE, app, cloud).
3. Seleccion de modelos por coste, latencia y calidad.
4. Gestion avanzada del contexto y estrategias de prompting.
5. Skills: diseno y reutilizacion en equipo.
6. Configuracion profesional: policies, approvals y sandbox.
7. Integraciones con MCP y herramientas externas.
8. Refactorizacion, revision de codigo y optimizacion.
9. Testing y debugging asistidos.
10. CI/CD y trabajo en equipos.
11. Seguridad, privacidad y control de calidad.
12. Limitaciones, mitigaciones y playbook corporativo.

## Como usarlo

Lee el plan completo y luego trabaja modulo por modulo. Cada modulo incluye teoria, buenas practicas, errores comunes y un laboratorio practico con resultado esperado.
